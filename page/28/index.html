4类LSA（ASBR-Summary LSA）

图示

内容

  1. 由ABR产生
  2. 在ABR所在的域内泛洪（不包括ASBR所在的域）
  3. 然后由其他的ABR重新生成
  4. 描述的是ASBR的路由信息

总结

5类LSA（AS External LSA）

图示

内容

  1. 有ASBR产生
  2. 在整个AS域内泛洪
  3. 描述的是AS外部引入的路由信息
  4. 不会重新生成LSA

LSA不同类型

Advertising Router

LSA Type LSA名称 Advertising Router 说明
1 Router LSA Router自己 Intra-Area Link
2 Network LSA DR Network Link
3 Network Summary LSA ABR Inter-Area Link
4 ASBR Summary LSA ABR ASBR Summary Link
5 AS External LSA ASBR AS External Link
6 All Routers Group Membership Link
7 NSSA External LSA ASBR NSSA AS External Link
8 External Attributes LSA
9 Opaque LSA（Link-Local）
10 Opaque LSA（Area-Local）
11 Opaque LSA（AS-Local）

Link State ID

LSA名称 Link State ID
Router-LSA 生成这条LSA的路由器的Router ID
Network-LSA 所描述网段上DR接口的IP地址
Network-Summary-LSA 所描述的目的网段的地址
ASBR-Summary-LSA 所描述的ASBR的Router ID
AS-External-LSA 所描述的目的网段的地址

传播范围

LSA类型 通告路由器 LSA内容 传播范围
Router LSA(Type-1) OSPF Router 拓扑信息+路由信息 本区域内
Network LSA(Type-2) DR 拓扑信息+路由信息 本区域内
Network Summary LSA(Type-3) ABR 域间路由信息 非（Totally）STUB区域
ASBR Summary LSA(Type 4) ABR ASBR的Router ID 非（Totally）STUB区域
AS External LSA（Type 5） ASBR 外部路由信息 非（Totally）STUB区域
NSSA LSA(Type 7) ASBR NSSA域外部路由信息 （Totally）NSSA区域
