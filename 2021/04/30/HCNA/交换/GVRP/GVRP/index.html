GARP

概述

  1. Generic Attribute Registration Protocol 通用属性注册协议

  2. 在交换机之间分发、传播、注册某种属性（VLAN属性、组播地址等）

  3. 用于大中型网络，用于提高网络管理效率

  4. 一种协议规范

  5. 图示

消息类型

图标

消息类型 备注
join 注册，端口注册VLAN
leave 注销，端口注销VLAN
leave all 端口注销所有VLAN（除了VLAN1）

GVRP

概述

  1. GARP VLAN Registation Protocol GARP VLAN注册协议

  2. GVRP基于GARP的工作机制，用来传输VLAN属性信息

  3. 图示

应用

  1. 接收来自其他交换机的VLAN注册信息，并动态更新本地的VLAN注册信息，包括当前的VLAN、VLAN成员等
  2. 将本地注册的VLAN信息向其他的交换机传播，为了使同一个交换网内的所有支持GVRP的设备VLAN信息一致
  3. 手动配置的是静态VLAN（common），通过GVRP注册的是动态VLAN (dynamic)
  4. GVRP传播的VLAN注册信息包括本地手动配置的静态VLAN信息和来自其他交换机的动态注册信息

GVRP消息类型

单向注册

单向注销

GVRP注册模式

normal

  1. 交换机默认是normal模式，允许动态和静态的VLAN注册

  2. 会同时发送动态和静态的VLAN信息

  3. 图示

forbidden

  1. 接口上不允许动态VLAN的注册

  2. 同时删除端口上的所有VLAN（除了VLAN1以外）

  3. 图示

fixed

  1. 端口不允许东涛VLAN信息的注册以及注销

  2. 端口只发送静态VLAN信息

  3. 图示
