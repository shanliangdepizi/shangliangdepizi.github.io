DHCP配置

图表

命令 备注
dhcp enable 开启DHCP功能
以下都是在接口模式下：：
dhcp select interface 关联接口与接口地址池
dhcp server dns-list 配置接口地址池的DNS服务器地址
dhcp server lease 数字 配置接口地址池的租期，默认为1天
dhcp server excluded-ip-address 地址 配置接口地址池的排除地址范围
dhcp server static-bind-ip-address
x.x.x.x mac-address xxx-xxxx-xxxx		 配置静态绑定
dhcp select global 关联接口和全局地址池
ip pool 名称 创建全局地址池
以下都是在 ip pool 下配置 配置全局地址池的详细配置
network 地址 可分配的网段地址
gateway-list地址 网关地址
dns-list 地址 DNS服务器地址
lease 数字 租期，默认是1天
excluded-ip-address 地址 排除的地址范围
查看配置命令
display ip pool [interface 接口名 all] 查看接口地址池的属性
dis ip pool interface vlanif20 used 查看接口地址池已使用的IP
PC端的命令（win系统）
ipconfig /release 释放租期
ipconfig /renew 重新获取
